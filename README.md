# SCILIVE - FRONTEND

## Tech Stack

- NextJS 13+
- TailwindCSS
- Typescript
- i18Next
- Redux Toolkit
- ImmerJS
- NextAuth

## Getting Started

First, install & run the development server:

```bash

yarn  install

yarn  dev

```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
